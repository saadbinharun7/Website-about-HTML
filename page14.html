list তৈরি করার জন্য:(ul দিয়ে)
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <ul style="list-style-type:curcle">
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ul>
                    </body>
                </html>
            
        
            এখানে <ul> ট্যাগটি unordered list তৈরি করে। এই কোডটি একটি তালিকা তৈরি করবে যেখানে প্রতিটি আইটেম একটি বুলেট পয়েন্ট দ্বারা চিহ্নিত হবে। এখানে list-style-type:circle ব্যবহার করা হয়েছে, যা বুলেট পয়েন্টকে একটি বৃত্তাকার চিহ্নে পরিবর্তন করে। list-style-type এর মান হতে পারে disc, square, circle ইত্যাদি।
            কোডটির ফলাফল এমন হবে: 
            
    
                
  • Item 1
    • 
                
  • Item 2
    • 
                
  • Item 3
    • 
            

        
list তৈরি করার জন্য:(ol দিয়ে)
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <ol type="i">
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ol>
                    </body>
                </html>
            
        
            এখানে <ol> ট্যাগটি ordered list তৈরি করে। এই কোডটি একটি তালিকা তৈরি করবে যেখানে প্রতিটি আইটেম একটি সংখ্যা দ্বারা চিহ্নিত হবে। এখানে type="i" ব্যবহার করা হয়েছে, যা রোমান সংখ্যা (i, ii, iii) ব্যবহার করে তালিকাটি তৈরি করে। type এর মান হতে পারে 1, a, A, i, I ইত্যাদি।
            কোডটির ফলাফল এমন হবে: 
            
    
                
  1. Item 1
    2. 
                
  2. Item 2
    3. 
                
  3. Item 3
    4. 
            

        
বিভিন্ন যাগা থেকে list তৈরি করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <ol type="A" start="4">
                            <li>Item 1</li>
                            <li>Item 2</li>
                            <li>Item 3</li>
                        </ol>
                    </body>
                </html>
            
        
            এখানে type="A" ব্যবহার করা হয়েছে, যা বড় হাতের ইংরেজি অক্ষর (A, B, C) ব্যবহার করে তালিকাটি তৈরি করে। start="4" ব্যবহার করা হয়েছে, যা তালিকাটির শুরু সংখ্যা 4 থেকে শুরু করবে। 
            কোডটির ফলাফল এমন হবে: 
            
    
                
  1. Item 1
    2. 
                
  2. Item 2
    3. 
                
  3. Item 3
    4. 
            

        
discription title তৈরি করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <dl>
                            <dt>Title 1</dt>
                            <dd>Description 1</dd>
                            <dt>Title 2</dt>
                            <dd>Description 2</dd>
                        </dl>
                    </body>
                </html>
            
        
            এখানে <dl> ট্যাগটি description list তৈরি করে। এই কোডটি একটি তালিকা তৈরি করবে যেখানে প্রতিটি আইটেমের জন্য একটি শিরোনাম (title) এবং তার সাথে সম্পর্কিত বর্ণনা (description) থাকবে। 
            কোডটির ফলাফল এমন হবে: 
            

                
Title 1

                
Description 1

                
Title 2

                
Description 2
