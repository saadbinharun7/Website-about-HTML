লাইন গেপ দিতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <p> This is a paragraph <br> and this is so big </p>
                    </body>
                </html>
            
        
 
            <br> ট্যাগ দিয়ে পরের লাইনে যাওয়া হয়।
            কোডটির ফলাফল এমন হবে: 
           this is a paragraph 
           and this is so big

        




একটি রো তৈরী করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <p> This is a paragraph <hr> and this is so big </p>
                    </body>
                </html>
            
        
 
            <hr> ট্যাগ দিয়ে রো তৈরী করা হয়।
            কোডটির ফলাফল এমন হবে: 
           this is a paragraph 
           and this is so big
