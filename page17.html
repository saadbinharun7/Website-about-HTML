code tag এর কাজ
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <p>This is a paragraph.</p>
                        <code>This is code.</code>
                    </body>
                </html>
            
        
            এখানে <code> ট্যাগটি কোডের অংশকে চিহ্নিত করতে ব্যবহৃত হয়। এটি সাধারণত প্রোগ্রামিং কোড বা কম্পিউটার সম্পর্কিত তথ্য প্রদর্শনের জন্য ব্যবহৃত হয়। 
            কোডটির ফলাফল এমন হবে: 
            
This is a paragraph.

            This is code.
        




pre tag এর কাজ
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <pre>This is preformatted text.</pre>
                    </body>
                </html>
            
        
            এখানে <pre> ট্যাগটি পূর্বনির্ধারিত টেক্সট প্রদর্শনের জন্য ব্যবহৃত হয়। এটি সাধারণত ফর্ম্যাট করা টেক্সট বা কোড ব্লক প্রদর্শনের জন্য ব্যবহৃত হয়, যেখানে স্পেস এবং লাইন ব্রেকগুলি সংরক্ষিত হয়। 
            কোডটির ফলাফল এমন হবে: 
            
This is preformatted text.

        




address tag এর কাজ
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <address>123 Main St, Anytown, USA</address>
                    </body>
                </html>
            
        
            এখানে <address> ট্যাগটি যোগাযোগের তথ্য প্রদর্শনের জন্য ব্যবহৃত হয়। এটি সাধারণত ঠিকানা, ফোন নম্বর, ইমেল ঠিকানা ইত্যাদি প্রদর্শনের জন্য ব্যবহৃত হয়। 
            কোডটির ফলাফল এমন হবে: 
            
123 Main St, Anytown, USA

                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

            কোডটির ফলাফল এমন হবে: 
            Ctrl + C

                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <samp>Sample output</samp>
                    </body>
                </html>
            
        
            এখানে <samp> ট্যাগটি প্রোগ্রাম আউটপুট প্রদর্শনের জন্য ব্যবহৃত হয়। এটি সাধারণত কম্পিউটার প্রোগ্রামের আউটপুট বা ফলাফল প্রদর্শনের জন্য ব্যবহৃত হয়। 
            কোডটির ফলাফল এমন হবে: 
            Sample output
        




var tag এর কাজ
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <var>x = 10</var>
                    </body>
                </html>
            
        
            এখানে <var> ট্যাগটি একটি ভেরিয়েবল বা পরিবর্তনশীল প্রদর্শনের জন্য ব্যবহৃত হয়। এটি সাধারণত প্রোগ্রামিং ভাষায় ভেরিয়েবল নাম প্রদর্শনের জন্য ব্যবহৃত হয়। 
            কোডটির ফলাফল এমন হবে: 
            x = 10
        




cite tag এর কাজ
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <cite>Shakespeare</cite>
                    </body>
                </html>
            
        
            এখানে <cite> ট্যাগটি একটি উদ্ধৃতি বা রেফারেন্স প্রদর্শনের জন্য ব্যবহৃত হয়। এটি সাধারণত বই, প্রবন্ধ, বা অন্যান্য উৎসের নাম প্রদর্শনের জন্য ব্যবহৃত হয়। 
            কোডটির ফলাফল এমন হবে: 
            Shakespeare
        




audio file আনতে
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <audio controls>
                            <source src="audio.mp3" type="audio/mp3">
                            Your browser does not support the audio element.
                        </audio>
                    </body>
                </html>
            
        
            এখানে <audio> ট্যাগটি অডিও ফাইল অন্তর্ভুক্ত করার জন্য ব্যবহৃত হয়। 
            controls অ্যাট্রিবিউটটি অডিও প্লেয়ার কন্ট্রোল প্রদর্শন করে। 
            source ট্যাগের src অ্যাট্রিবিউটে অডিও ফাইলের পাথ উল্লেখ করতে হয়। 
            কোডটির ফলাফল এমন হবে: 
            
        




video file আনতে
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <video width="320" height="240" controls>
                            <source src="video.mp4" type="video/mp4">
                            Your browser does not support the video tag.
                        </video>
                    </body>
                </html>
            
        
            এখানে <video> ট্যাগটি ভিডিও ফাইল অন্তর্ভুক্ত করার জন্য ব্যবহৃত হয়। 
            width এবং height অ্যাট্রিবিউটগুলি ভিডিওর আকার নির্ধারণ করে। 
            controls অ্যাট্রিবিউটটি ভিডিও প্লেয়ার কন্ট্রোল প্রদর্শন করে। 
            source ট্যাগের src অ্যাট্রিবিউটে ভিডিও ফাইলের পাথ উল্লেখ করতে হয়। 
            কোডটির ফলাফল এমন হবে:
