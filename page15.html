এক ওয়েবসাইটের ভিতরে অন্য ওয়েবসাইট দেখাতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <iframe src="https://www.freepik.com/" title="fp" style="height: 100px;width:200px;border:3px solid black"> </iframe>
                    </body>
                </html>
            
        
            এখানে <iframe> ট্যাগটি ব্যবহার করা হয়েছে যা একটি ওয়েব পেজের ভিতরে অন্য একটি ওয়েব পেজ দেখানোর জন্য ব্যবহৃত হয়।
            src অ্যাট্রিবিউটটি সেই URL নির্দেশ করে যা আপনি দেখতে চান। 
            title অ্যাট্রিবিউটটি iframe এর জন্য একটি শিরোনাম প্রদান করে, যা অ্যাক্সেসিবিলিটির জন্য গুরুত্বপূর্ণ। 
            style অ্যাট্রিবিউটটি iframe এর উচ্চতা, প্রস্থ এবং বর্ডার নির্ধারণ করে।
            কোডটির ফলাফল এমন হবে: 
            
        
 Head ট্যাগের বিভিন্ন ট্যাগ:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <title>My Web Page</title>
                        <meta charset="UTF-8">
                        <meta name="keywords" content="HTML, CSS, JS">
                        <meta name="description" content="This is ....">
                        <meta name="viewport" content="width = device-width; initial-scale = 10">
                    </head>

                    <body>
                        <p>Welcome to my web page!</p>
                    </body>
                </html>
            
        
            এখানে <head> ট্যাগের ভিতরে বিভিন্ন ট্যাগ ব্যবহার করা হয়েছে। 
            <title> ট্যাগটি পৃষ্ঠার শিরোনাম নির্ধারণ করে, যা ব্রাউজারের ট্যাবে প্রদর্শিত হয়। 
            <meta> ট্যাগগুলি পৃষ্ঠার মেটাডেটা প্রদান করে, যেমন চরিত্র সেট, কীওয়ার্ড, বর্ণনা এবং ভিউপোর্ট সেটিংস। 
            এই ট্যাগগুলি সার্চ ইঞ্জিন অপটিমাইজেশন (SEO) এবং রেস্পন্সিভ ডিজাইনের জন্য গুরুত্বপূর্ণ।
            এখানে <meta charset="UTF-8"> ট্যাগটি পৃষ্ঠার চরিত্র সেট নির্ধারণ করে, যা ইউনিকোড (UTF-8) ব্যবহার করে।
            <meta name="keywords" content="HTML, CSS, JS"> ট্যাগটি পৃষ্ঠার কীওয়ার্ড নির্ধারণ করে যা সার্চ ইঞ্জিনে পৃষ্ঠাটি খুঁজে পেতে সাহায্য করে।
            <meta name="description" content="This is ...."> ট্যাগটি পৃষ্ঠার বর্ণনা প্রদান করে, যা সার্চ ইঞ্জিনের ফলাফলে প্রদর্শিত হয়।
            <meta name="viewport" content="width=device-width, initial-scale=1.0"> ট্যাগটি পৃষ্ঠার ভিউপোর্ট সেটিংস নির্ধারণ করে, যা মোবাইল ডিভাইসে রেস্পন্সিভ ডিজাইন নিশ্চিত করে।
            কোডটির ফলাফল এমন হবে:
            
Welcome to my web page!
