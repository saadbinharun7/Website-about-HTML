table style করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <style>
                            table, th, td {
                                border: 1px solid black;
                                border-collapse: collapse;
                            }
                            th, td {
                                padding: 5px 15px;
                            }
                        </style>
                    </head>

                    <body>
                        <table>
                            <tr>
                                <th>Name</th>
                                <th>Roll no</th>
                                <th>Reg fee</th>
                            </tr>
                            <tr>
                                <td>Sohel rana</td>
                                <td>12</td>
                                <td>1000</td>
                            </tr>
                            <tr>
                                <td>Sohag rana</td>
                                <td>13</td>
                                <td>1000</td>
                            </tr>
                            <tr>
                                <td>Sohil rana</td>
                                <td>14</td>
                                <td>1000</td>
                            </tr>
                        </table>
                    </body>
                </html>
            
        
            head এর মধ্যে <style> ট্যাগ দিয়ে table, th, td এর style করতে হয়। 
            border: 1px solid black; দিয়ে table, th, td এর border তৈরি করা হয়। 
            border-collapse: collapse; দিয়ে table এর border collapse করা হয়, আর্থাৎ border এর মধ্যে gap থাকবে না।
            padding: 5px 15px; দিয়ে th, td এর padding তৈরি করা হয়। 5px vertical এবং 15px horizontal padding দেওয়া হয়েছে। padding হলো cell এর ভিতরের জায়গা।
        




collumn মার্জ করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <style>
                            table, th, td {
                                border: 1px solid black;
                                border-collapse: collapse;
                            }
                            th, td {
                                padding: 5px 15px;
                            }
                        </style>
                    </head>

                    <body>
                        <table>
                            <tr>
                                <th colspan="3">Student Information</th>
                            </tr>
                            <tr>
                                <th>Name</th>
                                <th>Roll no</th>
                                <th>Reg fee</th>
                            </tr>
                            <tr>
                                <td>Sohel rana</td>
                                <td>12</td>
                                <td>1000</td>
                            </tr>
                            <tr>
                                <td>Sohag rana</td>
                                <td>13</td>
                                <td>1000</td>
                            </tr>
                            <tr>
                                <td>Sohil rana</td>
                                <td>14</td>
                                <td>1000</td>
                            </tr>
                        </table>
                    </body>
                </html>
            
        
            Note: <th colspan="3"> দিয়ে column মার্জ করা হয়েছে। এখানে 3 column মার্জ করা হয়েছে। 
            colspan attribute দিয়ে column মার্জ করা হয়।
            কোডটির ফলাফল এমন হবে:
Student Information
Name Roll no Reg fee
Sohel rana 12 1000
Sohag rana 13 1000
Sohil rana 14 1000 




row মার্জ করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <style>
                            table, th, td {
                                border: 1px solid black;
                                border-collapse: collapse;
                            }
                            th, td {
                                padding: 5px 15px;
                            }
                        </style>
                    </head>

                    <body>
                        <table>
                            <tr>
                                <th rowspan="3">Name</th>
                                <th>Roll no</th>
                                <th>Reg fee</th>
                            </tr>
                            <tr>
                                <td>12</td>
                                <td>1000</td>
                            </tr>
                            <tr>
                                <td>13</td>
                                <td>1000</td>
                            </tr>
                        </table>
                    </body>
                </html>
            
        
            Note: <th rowspan="3"> দিয়ে row মার্জ করা হয়েছে। এখানে 3 row মার্জ করা হয়েছে। 
            rowspan attribute দিয়ে row মার্জ করা হয়।
            কোডটির ফলাফল এমন হবে:
Name Roll no Reg fee
12 1000
13 1000
