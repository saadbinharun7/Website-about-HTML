HTML Basics

HTML-এর মৌলিক গঠন নিম্নরূপ:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>

                    </body>
                </html>
            
        
 
            এখানে  <DOCTYPE html>  ডকুমেন্টের টাইপ হিসেবে HTML চিহ্নিত করে।
        
To write a title (যা নতুন ট্যাব এলাকায় দেখায়) 
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <title> example </title>
                    </head>

                    <body>

                    </body>
                </html>
            
        
            টাইটেল ট্যাগে  <title> </title>  যা লেখা থাকবে, তা ব্রাউজারের টাইটেল বারে দেখাবে।
        
To write a heading 
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <h1> First </h1>
                        <h2> Second </h2>
                        <h3> third </h3>
                        <h4> fourth </h4>
                        <h5> fifth </h5>
                        <h6> sixth </h6>
                    </body>
                </html>
            
        
            হেডিং ট্যাগ টেক্সটের সাইজ নির্ধারণ করে। এইচটিএমএলে ৬টি হেডিং ট্যাগ আছে: <h1> </h1>  সবচেয়ে বড় সাইজ , <h6> </h6> সবচেয়ে ছোট সাইজ.
