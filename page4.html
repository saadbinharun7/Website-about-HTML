ছবি বড় বা ছোট করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <img src = "name.jpg" alt = "natural image" style = " height: 50px; width: 100px">
                    </body>
                </html>
            
        
 
            এখানে  alt = " " এর ভিতরে natural image দিয়ে image টির ধরন উল্লেখ করা হয়েছে, যদি কোন কারনে image টি লোড না হয় তখন এটি দেখাবে।
            আাবার  style = " " এর ভিতরে ছবিটির দৈর্ঘ্য ও প্রস্থ pixel-এককে নির্ধারণ করে দেওয়া হয়েছে।

ছবির সাথে লিঙ্ক এড করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <a href = "https://www.youtube.com/" target = "_blank"">
                            <img src = "youtube.jpg">
                        </a>
                    </body>
                </html>
            
        
 
            এখানে এঙ্কর ট্যাগ দিয়ে লিঙ্ক এড করা হয়েছে, শুধু Click here এর পরিবর্তে ছবি এড করা হয়েছে।
