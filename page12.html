link decoration করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <style>
                            a:link {
                                text-decoration: none;
                                color: red;
                            }
                            a:visited {
                                text-decoration: none;
                                color: blue;
                            }
                            a:hover {
                                text-decoration: underline;
                                color: green;
                            }
                            a:active {
                                text-decoration: none;
                                color: orange;
                            }
                        </style>
                    </head>

                    <body>
                        <a href="www.facebook.com" target="_blank">This is a link</a>
                    </body>
                </html>
            
        
            এখানে link এর বিভিন্ন state এর জন্য CSS pseudo-classes ব্যবহার করা হয়েছে। 
            a:link - link এর default state।
            a:visited - link যখন visit করা হয়।
            a:hover - link যখন mouse pointer এর উপরে থাকে।
            a:active - link যখন click করা হয়।
            এই কোডটি link এর style পরিবর্তন করে এবং hover ও active অবস্থায় বিভিন্ন রঙ ও underline প্রয়োগ করে।
            কোডটির ফলাফল এমন হবে: This is a link
        
mail address তৈরি করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        
                    </head>

                    <body>
                        <a href="mailto:xyz@abc.com" target="_blank">Send Email</a>
                    </body>
                </html>
            
        
            এখানে mailto ব্যবহার করে একটি ইমেইল লিঙ্ক তৈরি করা হয়েছে। যখন ব্যবহারকারী এই লিঙ্কে ক্লিক করবে, তখন তাদের ডিফল্ট ইমেইল ক্লায়েন্ট খুলবে এবং একটি নতুন ইমেইল তৈরি করবে যার প্রাপক হবে নির্দিষ্ট করা ইমেইল ঠিকানা (xyz@abc.com)। এটি ওয়েবসাইটে সহজেই ইমেইল পাঠানোর সুবিধা দেয়।
            কোডটির ফলাফল এমন হবে: Send Email
