transparency এর জন্য:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <p style = "color: rgba(0,255,0,0.5)"> The quick brown fox jumps over the lazy dog </p>
                    </body>
                </html>
            
        
 
            এখানে rgba() ফাংশন ব্যবহার করা হয়েছে, যেখানে প্রথম তিনটি মান রঙের (Red, Green, Blue) এবং চতুর্থ মানটি স্বচ্ছতা (Alpha) নির্দেশ করে। alpha মান 0 থেকে 1 এর মধ্যে হতে পারে, যেখানে 0 সম্পূর্ণ স্বচ্ছ এবং 1 সম্পূর্ণ অস্বচ্ছ। এই ক্ষেত্রে, rgba(0,255,0,0.5) মানে হল সবুজ রঙের সাথে 50% স্বচ্ছতা।

              The quick brown fox jumps over the lazy dog 

        
hue দিয়ে color করতে:
            
                <!DOCTYPE html>
                <html>
                    <head>
                    
                    </head>

                    <body>
                        <p style = "color: hsl(0,50%,50%)"> The quick brown fox jumps over the lazy dog </p>
                    </body>
                </html>
            
        
 
            এখানে hsl() ফাংশন ব্যবহার করা হয়েছে, যেখানে প্রথম মানটি hue (0 থেকে 360 ডিগ্রি), দ্বিতীয় মানটি saturation (0% থেকে 100%), এবং তৃতীয় মানটি lightness (0% থেকে 100%) নির্দেশ করে। এই ক্ষেত্রে, hsl(0,50%,50%) মানে হল লাল রঙের সাথে 50% saturation এবং 50% lightness।

html এ color করতে website 
css হল cascading style sheet.
            css দিয়ে html এর element গুলোর style পরিবর্তন করা যায়। যেমন: color, font-size, background-color ইত্যাদি।
            css দিয়ে html এর element গুলোর position পরিবর্তন করা যায়। যেমন: top, bottom, left, right ইত্যাদি।
            css দিয়ে html এর element গুলোর display পরিবর্তন করা যায়। যেমন: block, inline, inline-block ইত্যাদি।
            
            css ৩ প্রকার:
            ১. inline css: html এর element এর মধ্যে style attribute ব্যবহার করে css লেখা হয়।
            ২. internal css: html এর head section এ style tag ব্যবহার করে css লেখা হয়।
            ৩. external css: html এর সাথে একটি css file যুক্ত করে css লেখা হয়। এই ক্ষেত্রে html এর head section এ link tag ব্যবহার করা হয়। external css এর জন্য একটি css file তৈরি করতে হয়। এই file এর extension .css হতে হবে। যেমন: style.css
        
inline css এর উদাহরন:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <style>
                            h2 {
                                color: red;
                                font-size: 20px;
                                background-color: yellow;
                            }
                        </style>
                    </head>

                    <body>
                        <h2> The quick brown fox jumps over the lazy dog </h2>
                    </body>
                </html>
            
        
 
            এখানে h2 ট্যাগের মধ্যে inline css ব্যবহার করা হয়েছে। এই css এর মাধ্যমে h2 ট্যাগের color, font-size এবং background-color পরিবর্তন করা হয়েছে।
            কোডটির ফলাফল এমন হবে: 
              The quick brown fox jumps over the lazy dog 

        
external css এর উদাহরন:
            
                <!DOCTYPE html>
                <html>
                    <head>
                        <link rel="stylesheet" type="text/css" href="style.css">
                    </head>

                    <body>
                        <h2> The quick brown fox jumps over the lazy dog </h2>
                    </body>
                </html>
            
        
 
            এখানে link tag ব্যবহার করে external css যুক্ত করা হয়েছে। এই css এর মাধ্যমে h2 ট্যাগের color, font-size এবং background-color পরিবর্তন করা হয়েছে। style.css ফাইলটি html ফাইলের সাথে একই ডিরেক্টরিতে থাকতে হবে।
